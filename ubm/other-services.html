Features Short Description SMS Syntax Example
Transaction Details Get transaction details for a transaction. You will receive a response SMS with the transaction details SHOW <Trans.id> SHOW 12345
Cancel ‘Viral Send’ Cancel the money transfer sent to a mobile number via ‘viral Send’ SENDCANCEL <Trans.id> SENDCANCEL 12345
Verify Agent Verify that the agent* is authorized and active. You will receive a SMS response AGENT <Agent OAN> AGENT 1234567
Change Mpin Change your existing Mpin. You would receive an IVR call promoting to enter a new pin CHGPIN CHGPIN
History Check history of last 5 transactions HIST HIST
Change Language Change the preferred language of communication (HINDI or ENGLISH) CHGLANG <Language> CHGLAND HINDI
Help Get list of all SMS commands HELP HELP
Help on Features Get SMS for help on a keywords HELP <Keywords> HELP SEND
Locate Agent Get details of all agents nearest to your area Agent <Your area pincode> Agent 400088
